# System design labs
A collection of backend engineering projects focused on understanding how modern software systems are designed, built, and scaled.  This repository documents my journey through system design fundamentals by implementing real-world inspired components and exploring the trade-offs behind different architectural decisions.

Each project is a standalone “lab” that demonstrates one core concept.
Together, they form a practical learning path toward becoming a strong backend/system design engineer.

### 🎯 Goals of This Repository
- Build real backend components used in production systems

- Understand how distributed systems behave under load

- Learn to design scalable APIs and services

- Practice architectural thinking and trade‑off analysis

- Explore modern tools like FastAPI, Redis, Docker, and message queues

- Create a portfolio of system‑design projects
